# Architectural Decisions

All architectural decisions records (ADRs) are saved in this folder. For new ADRs, copy over [template.md](template.md) to nnnn-title.md and modify.

The Markdown ADR (MADR) documentation is available at <https://adr.github.io/madr/> while general information about ADRs is available at <https://adr.github.io/>.
