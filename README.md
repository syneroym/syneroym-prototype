# syneroym
Cross-platform P2P app for autonomous cooperative value exchange

Syneroym is a cross platform app aimed at facilitating value exchange among autonomous cooperating community peers, typically individuals, groups, organizations. Peers can host services for themselves or community, interact with other community services, all through a set of micro-services and micro-apps (plugins, extensions) housed in this covering super app. Different variants of this superapp are available based on desired nature of use and installation platform capabilities.
