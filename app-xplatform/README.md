# syneroym web and mobile app

Cross-platform app that acts as a Syneroym peer

This is a Tauri based web and mobile cross-platform application that acts as a peer within the Syneroym ecosystem. Peers can host services for themselves or community, interact with other community services, through a set of micro-services and UI plugins (like micro-apps with UI) housed in this covering super app.

# Tauri + Vanilla TS
