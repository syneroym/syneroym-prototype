# Syneroym JavaScript Libraries (`lib-js`)

This directory contains shared JavaScript/TypeScript packages used primarily by the frontend applications (`app-xplatform`).

## Packages

- **`ui-solid`**: A component library built with [SolidJS](https://www.solidjs.com/). It contains reusable UI elements to ensure consistency across the Syneroym application.
- **`ui-styles`**: Contains shared configuration for styling, including [Tailwind CSS](https://tailwindcss.com/) configurations and global themes.
